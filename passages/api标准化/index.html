api标准化

1. Navigator

1.1 Webview 窗口

1.1.1 let ref = Navigator.open(url, target):Webview

打开新页面
  • ref: 当 target 为 ‘_blank’ 时，打开新窗口的对象
  • url: 新窗口打开的 url 地址
  • target:
    • _self: 在当前 Webview 重定向 url 地址 (暂时未使用)
    • _blank: 打开新的 Webview url 地址
    • _system: 打开系统默认浏览器新建窗口，指向 url 地址 (PC 下载错误提示去官网下载，打开官网)

    • 1.1.2 let webview = Webview.on(eventname, callback)

    增加 Webview 监听事件

    • eventname: {String}
      • enter: 进入 Webview
      • exit: Webview 关闭时触发
      • message: Webview 接受到信息 //参考浏览器原生 window.addEventListener(‘message’,function(params))
      • pause: Webview 隐藏
      • resume: Webview 页面再可见
    • callback: {Function}
      • params: {Object}
        • url: {String} Webview 的 url
        • data: {String|Object} 接受到的信息
        • origin: {String} 来源路径
        • timeStamp: {Number} 时间戳

    1.1.3 let webview = Webview.off(eventname,callback)

    移除 Webview 监听事件,不填写 callback 删除 eventname 所有回调

    1.1.4 Webview.postMessage(message)

    向自己发送消息，通过 message 监听

    • message: {Object|String} 发送信息 //参考浏览器原生 window.postMessage(message)

    1.1.5 Webview.close()

    关闭 Webview

    1.1.6 Webview.show()

    显示 Webview

    1.1.7 Webview.hide()

    隐藏 Webview

    1.2 Client 客户端

    1.2.1 let clientInfo = Navigator.getClientInfo()

    获取客户端信息

    • clientInfo: {Object}
      • version: {String} 3.0.0.184 版本号
      • platform: {String} ios,pc,android 平台
      • type: {String} student,teacher 教师，学生

    1.2.2 let userInfo = Navigator.getUserInfo()

    获取用户信息

    • userInfo: {Object}
      • userUUid: {String} ?
      • userId: {String} 用户ID
      • userName: {String} 用户姓名
      • realName: {String} 真实姓名
      • imgUrl: {String} 用户头像

    1.2.3 let courseId = Navigator.getCourseId()

    获取课程ID

    • courseId: {String} 课程ID

    1.2.4 let isAuthed = Navigator.isAuthed()

    用户是否登录

    • isAuthed: {Boolean} 是否登录

    1.2.5 Navigator.disableSlide(left_state, right_state)

    禁止页面左滑或右滑

    • left_state: {Number} 左滑 0 激活 1 禁用
    • right_state: {Number} 右滑 0 激活 1 禁用

    1.2.6 Navigator.disableRefresh(state)

    禁用下拉刷新

    • state: {Number} 0 激活 1 禁用

    1.2.7 Navigator.updateNavigationTitle(text)

    修改客户端原生标题

    • text: {String} 标题内容

    1.2.8 Navigator.allClose()

    关闭所有 Webview （回到书架）

    1.2.9 Navigator.returnIndex()

    返回首页（目录页）

    1.2.10 Navigator.close()

    关闭当前页

    1.2.11 Navigator.login()

    登录

    1.2.12 Navigator.buyBook(productId)

    购买产品

    • productId: {String} 产品ID

    1.2.13 Navigator.share({type, title, desc, url, imgUrl})

    分享

    • type: {Number} // 1 好友 2 朋友圈
    • title: {String} // 标题
    • desc: {String} // 简要描述
    • url: {String} // 要分享的链接
    • imgUrl: {String} // 图标地址

    4.2 Recorder

    4.2.1 Recorder.record({net, record, partType})

    录音

    • net: {String|Array} 评分net文件，不传录音时代表不评分
    • record: {record} 录音文件名称
    • partType: {Number} 0-partA;1-partB;2-partC;3-只录音，不评分

    4.2.2 Recorder.stop()

    录音停止

    4.2.3 let time = Recorder.getTime()

    获取录音时间

    • time: {Number} 当前录制时间

    4.2.4 let record = Recorder.on(eventname, callback)

    • eventname: {String}
      • load: 加载
      • start: 开始录音
      • running: 录制中
      • test: 评分
      • stop: 停止
      • error: 错误
      • update: 频谱更新
    • callback: {Function}
      • params: {Object}
        • code: {Number} 错误代码 – error 事件
        • message: {String} 错误信息 – error 事件
        • result: {Object} 结果 – stop 事件
          • score: {Number} 评分
          • accuracy: {Number} 精确度
          • completion: {Number} 完整度
          • fluency: {Number} 流畅度
          • sentence: {Array} 返回文本
          • phoneMap: {Object} 音素
          • record_url: {String} 阿里云地址
          • duration: {Number} 录音时长
          • recordName: {String} 录音名称
        • fftData: {Array(Number)} 频谱数据

    4.2.5 let record = Recorder.off(eventname, callback)

    4.2.6 Recorder.isRecording():Boolean

    正在录音

    4.2.7 Recorder.isTesting():Boolean

    正在评分

    4.2.8 Recorder.isLoading():Boolean

    正在加载

    5. NetWork

    5.1 HttpClient

    5.1.1 Promise:HttpClient.post(url, {type, params}).then(callback).call(errFn)

    • url: {String} url 地址
    • type: {Number} 1 实时请求， 2 闲时请求
    • params: {Object} 请求数据
    • callback: {Function}
      • response: {Object}
        • code: 状态码
        • data: 数据
        • msg: 信息
    • errFn: {Function}
      • error: {Object}
        • code: 状态码
        • msg: 信息

    6 Logger

    6.1 Logger.error(…args)

    影响到程序正常运行、当前请求正常运行的异常情况

    6.2 Logger.warn(…args)

    不应该出现但是不影响程序、当前请求正常运行的异常情况

    6.3 Logger.info(…args)

    系统运行信息,外部接口部分

    6.4 Logger.debug(…args)

    主要是在开发期间使用或者当系统出现问题后开发人员介入调试的时候用的，需要有助于提供详细的信息

    • args {Array}
      • arg {String|Object} // Object 将自动转化为 JSON 字符串， 所有的字符串用空格连接

    7.2 EventBus.recieveMessage(params)

    • data: {String|Object} 收到的信息
    • origin: {String} 信息来源

    8 FlipBook

    8.1 let bookInfo:Object = FlipBook.getBookInfo()

    • bookInfo: {Object}
      • bookId: {String}
      • bookName: {String}
      • isFree: {Number} 0 不可用 1 可用
      • authDate: {Number} 有效期，剩余天数
      • pruductId: {String} 产品ID(购买书可用)
      • studyPro: {String} 整本书的学习进度
      • price: {String} 价格

    8.2 let catalogNode:Array = FlipBook.getCatalogNode()

    • catalogNode: {Array(Object)}
      • node: {Object}
        • id: {String}
        • displayName: {String}
        • order: {Number} 序号
        • isContent: {Boolean} 是否是内容节点
        • isFree: {Number} 0 不免费 1 免费
        • parentId: {String} 父节点
        • lockState: {Number} 锁状态 0 解锁 1 已锁
        • isPlaned: {Number} 是否配置教学计划: 0 未配置 1 已配置
        • beginTime: {Number} 教学计划中的开始时间的时间戳
        • endTime: {Number} 教学计划中的结束时间的时间戳
        • publishType: {Number} 成绩公布方式 1 实时 2 时间结束时 3 手动
        • isPublished: {Number} 0 未发布 1 已发布
        • chapterState: {Number} 0 未下载 1 等待下载 2 已下载 3 更新
        • type: {Number} 1 章节 2 页面
        • welcomeFile: {String} 欢迎页
        • studyPro: {String} 章节学习进度
        • chapterPoster: {String} 章节封面图
        • extendParams: {Object} 扩展字段
          • knowledgeId: {String} 关联的知识点，用于专项练习
          • isSimilar: {Number} 是否有同类题，1 有 0 没有
          • extend1: {String} 预留扩展字段1
          • extend2: {String} 预留扩展字段2
          • extend3: {String} 预留扩展字段3

    8.3 FlipBook.on(eventname, callback)

    • eventname: {String}
      • download
    • callback: {Function}
      • data: {Object} 下载返回的结果
        • chapterId: {String} 章节ID
        • state: {Number} 0 下载 1 解压 2 下载失败 3 解压失败 4 解压成功
        • progress: {String} 下载进度

    8.4 FlipBook.off(eventname, callback)

    8.5 FlipBook.submitEleAttr(arr)

    提交每道题作答数据

    • arr: {Array(Object)}
      • elementId: {String}
      • elementType: {Number}
      • elementAttr: {Object}
        • question_id: {String}
        • question_type: {Number}
        • order: {Number}
        • user_score: {Number}
        • user_answer: {String}
        • result: {Number}
        • record_url: {String}

    8.6 let data = FlipBook.getTasksUploadStatus(taskId?:{String})

    获取客户端上传队列中还未上传成功的task信息

    • 指定taskId *
    • data: {Object}
      • code: {Number} 反馈代码
      • message: {String} 反馈信息
      • status: {Number} 状态 -1 失败 0 上传中 1 上传成功
      • taskId: {String} taskId
      • max: {Number} 上传数据总大小
      • progress: {Number} 已上传数据大小
      • percent: {Number} 上传进度 0~100
    • taskMap *
    • data: {Object}
      • ${taskId}: {Object}
        • code: {Number} 反馈代码
        • message: {String} 反馈信息
        • status: {Number} 状态 -1 失败 0 上传中 1 上传成功
        • taskId: {String} taskId
        • max: {Number} 上传数据总大小
        • progress: {Number} 已上传数据大小
        • percent: {Number} 上传进度 0~100

    8.7 FlipBook.submitTask(data)

    提交 task 数据

    • data: {Object}
      • taskId: {String} task ID
      • taskNo: {Number} task 数目
      • score: {Number} task 成绩
      • chapterId: {String} 章节ID
      • percent: {Number} task 完成率
      • extendParams: {Object}
        • phonemes: {Array(Object)}
          • phoneme: {Object}
            • phonemeId: {String} 音素 ID
            • score: {Number} 0~100 最多保留2位小数
          • spokens: {Array(Object)} 参考 互动听说-读一读
            • recordUrl: {String} 录音文件地址
            • desc: {String} 描述信息
            • score: {Number} 分数
          • countCorrectQuestions: {Number} 是否统计试题做对人数，1 表示统计 参考 每周探索 或者 互动听说-听一听

    8.8 FlipBook.deleteChapter(chapterId)

    删除章节

    • chapterId: {String} 章节ID

    8.9 let pageInfo = FlipBook.loadPageInfo()

    页面信息

    • pageInfo: {Object}
      • chapterId: {String}
      • pageId: {String}
      • pageNum: {String}
      • displayName: {String}
      • fullChapterName: {Array} 完整的章节名称，数组下标从零开始表示最外层章节名称到最里面章节名称

    8.10 let chapterInfo = FlipBook.loadChapterInfo()

    章节信息

    • chapterInfo: {Object}
      • chapterId: {String} 章节ID
      • chapterName: {String} 章节名称
      • order: {Number} 序号
      • parentId: {String} 父节点
      • tasks: {Array(Object)}
        • task: {Object}
          • pages: {Array(Object)} 包含的页面ID
            • page: {Object}
              • pageId: {String} 页面ID
          • score: {Number} task 成绩
          • taskId: {String} taskId
          • taskName: {String} task 名称
          • taskNum: {Number} task 数目
          • percent: {Number} 0~100 完成率
          • isSubmitFailed: {Number} 是否提交失败
          • totalScore: {Number} 总分

    8.11 let taskInfo = FlipBook.loadTaskInfo()

    task 信息

    • taskInfo: {Object}
      • chapterId: {String}
      • taskId: {String}
      • order: {Number}
      • taskName: {String}
      • pages: {Array(Object)} 包含的页面ID
        • page: {Object}
          • pageId: {String} 页面ID
      • lockState: {Number} 锁状态 1 锁定 0 解锁
      • isPlaned: {Number} 是否配置了教学计划: 0 未配置 1 已配置
      • beginTime: {Number} 教学计划中的开始时间时间戳
      • endTime: {Number} 教学计划中的结束时间时间戳
      • publishType: {Number} 成绩公布方式: 1 实时 2 时间结束 3 手动
      • isPublished: {Number} 是否发布: 0 未发布 1 已发布

    8.12 let answerMark = FlipBook.getAnswerMark(taskId)

    获取作答信息

    • taskId: {String} task ID
    • answerMark: {Object} 作答信息
      • ${questionId}: {Object} 小题的 Id 为键值
        • elementList: {Array(Object)} 元素列表
          • element: {Object} 小题元素
            • elementAttr: {Object} 小题元素具体的信息
              • answer: {String} 答案
              • duration: {Number} 录音时长
              • order: {Number} 顺序
              • question_id: {String}
              • question_type: {Number} 题型
              • result: {Number} -1 0 1
              • score: {Number} 成绩
              • timestamp: {Number} 时间戳
              • user_answer: {String} 用户答案
              • user_score: {Number} 用户成绩
              • wrongnote_flag: {Number}
            • elementId: {String} 元素ID
            • elementType: {Number} 元素类型

    8.13 FlipBook.saveAnswerMark(taskId, answerMark)

    保存作答信息
